# Hardware

Motor controllers, sensors, and operator interfaces. All motor IDs are CAN IDs.
Motor configuration (gear ratios, feedback sensors, current limits) is set via
Phoenix Tuner X and stored in `src/main/java/frc/robot/generated/TunerConstants.java`.

---

## Drivetrain

**Type:** 4-module swerve (holonomic)

Each module has:
- 1 Talon FX (Kraken X60) — drive motor, `VelocityTorqueCurrentFOC`
- 1 Talon FX (Kraken X60) — turn motor, `PositionVoltage`
- 1 CANcoder — absolute turn encoder (fused into Talon FX feedback)

Module CAN IDs and encoder offsets: see `generated/TunerConstants.java` (auto-generated by
Phoenix Tuner X, do not edit manually).

Module geometry (from `DriveConstants`):
- Front offset: 11 inches from center
- Side offset: 11 inches from center
- Module positions: FL, FR, BL, BR

**Gyro:** Pigeon 2 (primary) via `GyroIOPigeon2`. NavX fallback via `GyroIONavX`.
Gyro instantiation selects based on `Constants.robotType`.

**CAN bus:** CAN FD (if available) at 250 Hz odometry, standard CAN at 100 Hz.
Bus name configured in `TunerConstants`.

## Cameras

4 PhotonVision cameras connected over network to the roboRIO. Camera transforms
are defined in `VisionConstants.java`.

| Index | Name | X (fwd) | Y (left) | Z (up) | Pitch |
|-------|------|---------|---------|--------|-------|
| 0 | RobotRight | 1.0" | −12.17" | 6.44" | −15° |
| 1 | RobotLeft | 1.0" | +12.42" | 6.44" | −15° |
| 2 | ShooterRight | −13.58" | −6.13" | 12.53" | −15° |
| 3 | ShooterLeft | −13.58" | +6.13" | 12.53" | −15° |

All cameras pitched −15° (lens angled up toward AprilTags on field structures).

PhotonVision runs on a coprocessor. Communication is over NetworkTables via
the standard PhotonVision NT protocol. The `VisionIOPhotonVision` implementation
wraps a `PhotonCamera` object per camera.

## Operator Interface

Defined in `Triggers.java` and `HardwareConstants.ControllerConstants`.

**Port 0 — Thrustmaster HOTAS joystick (primary driver):**

| Axis | Function |
|------|----------|
| 0 (X) | Strafe left/right |
| 1 (Y) | Drive forward/backward |
| 2 (Twist) | Rotate |

| Button | Function |
|--------|----------|
| 1 | Shoot (primary trigger) |
| 2 | Align for trench |
| 3 | Intake retract (pivot up) |
| 4 | Intake deploy (pivot down) |
| 5 | Run intake rollers |
| 6 | Manual compress |
| 8 | Align for bump zone |
| 10 | Tower shot (fixed angle/velocity) |
| 11 | Pass shot |

**Port 1 — Xbox controller (backup / operator):**

| Button | Function |
|--------|----------|
| X | X-lock wheels |
| B | Shoot (simulation only) |
| Y | Sim intake controls |
| POV Up/Down | Sim intake controls |

**Port 2 — Xbox controller (simulation only):**

Used only when `Constants.currentMode == SIM` for testing without the primary
joystick.

## Constants Reference

All operator and hardware constants are in `HardwareConstants.java`.

### Voltages

| Constant | Value | Used by |
|----------|-------|---------|
| `transportVoltage` | −7 V | Transport |
| `intakeVoltage` | +12 V | Intake roller |
| `agitateVoltage` | +3 V | Intake roller (slow eject) |
| `prestageIdleVoltage` | 8 V | Prestage idle mode |

### Velocities

| Constant | Value | Used by |
|----------|-------|---------|
| `prestageShootVelocity` | 3000 RPM | Prestage during shot |
| `prestageIdleVelocity` | 1300 RPM | Prestage default |
| `feederVelocity` | −3000 RPM | Upper and lower feeder |
| `transportVelocity` | −1800 RPM | Transport |

### Thresholds

| Constant | Value | Meaning |
|----------|-------|---------|
| `flywheelSpinupThreshold` | 200 RPM | "Spun up" tolerance |
| `hubAlignmentThreshold` | 1.5° | Tight alignment for shooting |
| `looseAlignmentThreshold` | 7.0° | Loose alignment (wider gate) |

### Timing

| Constant | Value | Usage |
|----------|-------|-------|
| `spinupTimeout` | 0.5 s | Max wait for flywheel spinup before feeding |
| `alignmentTimeout` | 1.5 s | Max wait for heading alignment |
| `compressDelay` | 0.5 s | Delay before first compress |
| `compressBetweenDelay` | 0.15 s | Delay between jostle steps |

### Fixed Shot Constants

**Tower shot (`TowerConstants`):**
- Flywheel: 1625 RPM
- Hood: 2.5°

**Pass shot (`PassConstants`):**
- Flywheel: 2700 RPM
- Hood: 35°
